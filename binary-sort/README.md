## 二分问题

二分问题指的是:

> 把一段区间一分为二，一半区间满足某种性质，另一半不满足。此时可以用二分法寻找边界。

思想:

> 在一个区间内部二分答案，每次选答案所在的区间进行处理。当区间为1的时候，就是答案。

二分模板分为整数模板和浮点数模板,其中整数模板有两种

整数二分模板：
```go
// 模板一
l, r := 0, len(nums)-1
for l < r {
    mid := (l + r) >> 1
    if nums[mid] >= target {
        r = mid
    } else {
        l = mid + 1
    }
}

// 模板二
l, r := 0, len(nums)-1
for l < r {
    mid := (l + r + 1) >> 1
    if nums[mid] <= target {
        l = mid
	} else {
        r = mid - 1
    }  
}
```
